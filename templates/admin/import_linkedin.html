Import via ligne de commande

Vous pouvez également déclencher l'import directement depuis le terminal :

# Import standard
python manage.py import_linkedin_jobs

# Import personnalisé
python manage.py import_linkedin_jobs --query "stage PFE informatique" --location "Casablanca" --max-results 30

# Voir toutes les options
python manage.py import_linkedin_jobs --help
